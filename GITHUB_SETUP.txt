# Quick GitHub Setup (Optional but Recommended)

If you want to use GitHub integration for easier deployment:

1. Go to https://github.com and create account
2. Click "New repository"
3. Name it "pdf-tools-suite" or whatever you want
4. Make it Public or Private
5. DON'T check "Add a README file"
6. Click "Create repository"

7. Copy these commands and run in terminal:
```
git init
git add .
git commit -m "Initial commit - PDF Tools Suite with PWA support"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/YOUR_REPO_NAME.git
git push -u origin main
```

Replace YOUR_USERNAME and YOUR_REPO_NAME with your actual GitHub details.

Then you can deploy directly from GitHub to Vercel/Railway/Render!
